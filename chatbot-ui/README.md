# AcreetionOS Chatbot UI

A modern chatbot UI for AcreetionOS, GenesisOS, TornadoOS, EcoOS, and StormOS knowledge. Uses React and OpenAI API for internet access.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Add your OpenAI API key in `src/Chatbot.js` (replace `YOUR_OPENAI_API_KEY`).
3. Start the development server:
   ```bash
   npm start
   ```

## Features
- Chatbot appears on the right side of the app
- Styled with modern UI
- Answers questions about the listed OSes
- Uses OpenAI API for internet-connected responses
